Scripting

Mmm so the title of this page is "coding" but really it should be "scripting" hehe. As time passes by I’ll upload some small tools and scripts I’ve developed. Since I made these for my own personal use the mileage you get out of them may vary. Also it should be noted that I have some sick and twisted liking for Bash (though I might release some python code as well…). These scripts are all developed on various Backtrack/Kali Linux builds, but if you have some motivation I’m sure you can get them to work on other distributions as well (provided that you install the necessary tools). If anyone has any suggestions or remarks about these scripts, feel free to get in touch…
